Projeto feito com finalidades educacionais para adquirir novos conhecimentos e por em prática os quais eu já possuo.

- Projeto simples para supermecado, padarias entre outros. Como um simples sistema de gerenciamento, tanto de clientes quanto de fornecedor e produtos.

- Este projeto possui um módulo principal e três módulos secundários que auxiliam o primeiro.

- Algumas partes do projeto foram reaproveitadas de trabalhos anteriores e alguns códigos da internet no qual me auxiliaram a construir coisas que eu ainda não tinha conhecimento.
